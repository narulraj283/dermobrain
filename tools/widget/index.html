Free Embeddable Widget for Dermatologists

Add DermoBrain's free widget to your practice website. It displays your DermoBrain profile, educational content, and helps build trust with patients — while earning you a valuable backlink.

How It Works

Simply add one line of code to your website. The widget automatically pulls your profile from DermoBrain and displays it in a clean, professional format that matches your website's style.

Installation

Copy and paste this code into your website's HTML where you want the widget to appear:

<script src="https://dermobrain.com/assets/js/dermobrain-widget.js" data-npi="YOUR_NPI_NUMBER"></script>

Replace YOUR_NPI_NUMBER with your 10-digit NPI number.

Widget Features

  • Your Profile — Displays your name, credentials, practice info, and specialties
  • Educational Content — Shows relevant dermatology articles for patient education
  • Appointment Link — Drives patients to your DermoBrain profile page
  • Responsive Design — Looks great on desktop, tablet, and mobile
  • SEO Benefit — Provides a dofollow backlink from dermobrain.com to your site

Customization Options

You can customize the widget with these optional attributes:

  • data-theme="light" or data-theme="dark" — Set color theme
  • data-width="400" — Set widget width in pixels
  • data-articles="3" — Number of educational articles to show
